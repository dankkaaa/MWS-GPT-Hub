# Сравнение Моделей v1

Этот документ нужен для выбора финальной текстовой модели для MVP.

## Цель

Сравнить:

- `mws-gpt-alpha`
- `qwen2.5-72b-instruct`

И понять:

- какую модель использовать как `default_chat`
- какую модель использовать как `strong_chat`

## Какие кейсы сравнивать

Для сравнения достаточно использовать:

- `EC-01` Обычный текстовый чат
- `EC-02` Follow-up с памятью
- `EC-03` Вопрос по файлу
- `EC-05` Summary по ссылке
- `EC-06` Web search для свежей информации

Описание этих кейсов находится в [EVAL_CASES.md](/Users/user/Desktop/MTS_mvp/EVAL_CASES.md).

## Что оценивать

Для каждого кейса нужно оценить:

- качество ответа
- краткость и ясность
- устойчивость к контексту
- честность при нехватке данных
- скорость ответа

## Шкала оценки

Можно использовать шкалу от `1` до `5`:

- `1` — плохо
- `2` — ниже ожиданий
- `3` — приемлемо
- `4` — хорошо
- `5` — отлично

## Правило выбора

Предварительное правило для MVP:

- если `mws-gpt-alpha` заметно быстрее и при этом не сильно уступает по качеству, оставить его как `default_chat`
- если `qwen2.5-72b-instruct` заметно лучше на задачах с контекстом, оставить его как `strong_chat`

- `default_chat_model = ...`
